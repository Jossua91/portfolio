Stage 1ère année BTS SIO (mai/juin 2023)


J’ai effectué mon stage de première année de BTS SIO avec ma professeure de développement au sein de l’établissement. Cette expérience m’a permis de travailler sur un projet concret dans un cadre encadré, mais très formateur.

L'objectif principal de mon stage était de développer une application web de gestion d’événements. Cette application devait permettre l’ajout, la modification et la suppression d’événements à travers une interface simple et fonctionnelle, qui devait être connectée à une base de données pour assurer la persistance des données.

Pour réaliser ce projet, j’ai approfondi un langage majeur du développement web moderne : React.js, et j'ai découvert Node.js pour le back-end. J’ai ainsi pu apprendre à structurer un projet avec ces outils, créer des composants React, gérer les états (state) et effectuer des appels API vers un serveur Node, qui lui-même interagissait avec une base de données (SQL). Afin de mettre en ligne ce projet sur un hébergement web, j'ai crée les mêmes appels API que j'avais crées avec Node.js mais en PHP.

Cette application m’a permis de :

  • - Comprendre la logique d’une architecture full-stack (React + Node.js + BDD)
  • - Créer une interface utilisateur dynamique avec React
  • - Développer une API REST basique en Node.js ainsi qu'en PHP
  • - Effectuer des opérations CRUD (Create, Read, Update, Delete)


J’ai pu faire mes premiers pas dans un environnement de développement complet, appris à structurer mon code, à raisonner comme un développeur, et à résoudre des problèmes concrets sur un projet de A à Z. Même si j’étais encadré, j’ai fait preuve d’autonomie dans la réalisation de mes tâches. J’ai appris à chercher, à tester et à corriger par moi-même, ce qui a renforcé ma motivation à poursuivre dans le domaine du développement web.

Compétences couvertes :

      • - Traiter des demandes concernant les applications : Création d’une application sur mesure à partir d’un besoin exprimé (gestion d’événements).
  • Travailler en mode projet :
      • - Analyser les objectifs et les modalités d’organisation d’un projet : Projet encadré mais avec autonomie dans l’analyse du besoin et la définition des fonctionnalités.

  • Organiser son développement professionnel :
      • - Mettre en place son environnement d’apprentissage personnel : Découverte et usage de React.js, Node.js, base de données SQL, gestion de projet full-stack.
      • - Développer son projet professionnel : Immersion dans un environnement concret, renforcement de l’autonomie, vision du métier de développeur web.
