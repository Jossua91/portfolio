Atelier de professionnalisation N°1
2ème année BTS SIO (Mars 2025)

Cet atelier s’inscrit dans la continuité d’une application existante développée sous Symfony pour un réseau de médiathèques. Elle permet de lister et gérer les formations proposées. La demande consistait à faire évoluer l’application en y ajoutant de nouvelles fonctionnalités, en modifiant certaines pages existantes, et en complétant la partie administrative de l'outil.

🔧 Langages et technologies utilisés :

  • PHP (Symfony)
  • Twig
  • JavaScript
  • HTML / CSS
  • MySQL
  • Git / GitHub
